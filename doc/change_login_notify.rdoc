= Documentation for Change Login Notify Feature

The change login notify feature emails the user when their login
is changed using the change login feature.

== Auth Value Methods

login_changed_email_body :: Body to use for the login changed emails
login_changed_email_subject :: Subject to use for the login changed emails

== Auth Methods

create_login_changed_email :: A Mail::Message for the login changed email to send.
send_login_changed_email :: Send the login changed email.
